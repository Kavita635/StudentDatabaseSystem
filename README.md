# Student Database System

This repository contains a simple Student Database Management System developed using the C programming language.

---

## Description

The Student Database System is a console-based application that allows users to manage student records efficiently.  
It supports basic operations such as adding, viewing, searching, updating, and deleting student information using file handling.

---

## Project Structure

StudentDatabaseSystem  
│  
├── student_db.c  
├── student_db  
├── data.txt  
├── README.md  

---

## Features

- Add new student records  
- View all stored student data  
- Search student details by roll number  
- Update existing student information  
- Delete student records  
- Persistent storage using file handling
